Create a new file test_stocks_strategies.py in the project root.
This is a standalone test and diagnostic tool that verifies
Stocks Algo 1 and Stocks Algo 2 are working correctly and can
cycle through each symbol in their universe.

─────────────────────────────────────────────────────────────────
OVERVIEW
─────────────────────────────────────────────────────────────────

The test suite must verify:

1. FCSAPI connectivity for stock symbols using NASDAQ: prefix
2. NDX market filter (SMA200) is computing correctly
3. Stocks Algo 1 can fetch candles and compute momentum scores
   for every symbol in its universe
4. Stocks Algo 2 can fetch candles and detect death crosses
   for every symbol in its universe
5. The monthly cycle (run_monthly) executes without errors
6. The daily cycle (run_daily) executes without errors
7. Exit checks (check_exits) execute without errors
8. The NASDAQ sync (sync_nasdaq100_symbols) works correctly
9. Full end-to-end signal generation test on a small subset

─────────────────────────────────────────────────────────────────
CLI ARGUMENTS
─────────────────────────────────────────────────────────────────

  --test          Which test to run. Options:
                    connectivity   Test FCSAPI for all symbols
                    ndx            Test NDX market filter
                    algo1          Test Stocks Algo 1 full cycle
                    algo2          Test Stocks Algo 2 full cycle
                    exits          Test exit checks for both
                    sync           Test NASDAQ 100 sync
                    all            Run all tests (default)

  --symbol        Test a single symbol only
                  e.g. --symbol AAPL

  --limit         Max symbols to test per strategy
                  Default: 10 (test first 10 symbols)
                  Use --limit 0 for all symbols

  --dry-run       Run all checks but do not insert any signals
                  or modify the database

  --verbose       Show full candle data and indicator values
                  for each symbol

  --fix           Attempt to auto-fix issues found during tests
                  e.g. re-verify unverified symbols,
                  refresh stale candle data

Examples:
  python3 test_stocks_strategies.py
  python3 test_stocks_strategies.py --test connectivity
  python3 test_stocks_strategies.py --test algo1 --limit 5
  python3 test_stocks_strategies.py --symbol AAPL --verbose
  python3 test_stocks_strategies.py --test all --dry-run
  python3 test_stocks_strategies.py --test algo1 --limit 0

Create a TestResults class to track pass/fail/warn per test:

class TestResults:
    def __init__(self):
        self.passed   = []
        self.failed   = []
        self.warnings = []
        self.skipped  = []

    def ok(self, name, detail=""):
        self.passed.append((name, detail))
        logger.info(f"  ✓ PASS | {name}" +
                    (f" | {detail}" if detail else ""))

    def fail(self, name, detail=""):
        self.failed.append((name, detail))
        logger.error(f"  ✗ FAIL | {name}" +
                     (f" | {detail}" if detail else ""))

    def warn(self, name, detail=""):
        self.warnings.append((name, detail))
        logger.warning(f"  ⚠ WARN | {name}" +
                       (f" | {detail}" if detail else ""))

    def skip(self, name, detail=""):
        self.skipped.append((name, detail))
        logger.info(f"  ○ SKIP | {name}" +
                    (f" | {detail}" if detail else ""))

    def summary(self):
        total = (len(self.passed) + len(self.failed) +
                 len(self.warnings) + len(self.skipped))
        print("\n" + "="*60)
        print(f"TEST SUMMARY")
        print("="*60)
        print(f"  ✓ Passed  : {len(self.passed)}")
        print(f"  ✗ Failed  : {len(self.failed)}")
        print(f"  ⚠ Warnings: {len(self.warnings)}")
        print(f"  ○ Skipped : {len(self.skipped)}")
        print(f"  Total     : {total}")
        print("="*60)
        if self.failed:
            print("\nFAILED TESTS:")
            for name, detail in self.failed:
                print(f"  ✗ {name}")
                if detail:
                    print(f"    → {detail}")
        if self.warnings:
            print("\nWARNINGS:")
            for name, detail in self.warnings:
                print(f"  ⚠ {name}")
                if detail:
                    print(f"    → {detail}")
        print()
        return len(self.failed) == 0
