Read STAGING_DB_PATH from environment variables — this should be the full absolute path to the production database file
If ENVIRONMENT is "staging" AND STAGING_DB_PATH is set, use STAGING_DB_PATH as the database path
Otherwise use the existing default path: os.path.join(os.path.dirname(__file__), "trading_data.db")
DATABASE_URL should still respect TRADING_ENGINE_DB_URL env var as the highest priority override, falling back to the computed path

TRADING_ENGINE_DB_URL  (highest — explicit full URL override)
STAGING_DB_PATH        (staging environment pointing to prod DB)
default path           (trading_engine/trading_data.db)
